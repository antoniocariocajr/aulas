## Guia de Estudos Kotlin

Este diretório contém um guia de estudos para a linguagem Kotlin.

A estrutura e o conteúdo dos arquivos aqui presentes foram criados para espelhar o formato do guia de estudos de Java, localizado no diretório `/Java`. Essa abordagem foi adotada intencionalmente para proporcionar uma experiência de aprendizado consistente e facilitar a transição ou o estudo comparativo entre as duas linguagens para os usuários familiarizados com este repositório.
