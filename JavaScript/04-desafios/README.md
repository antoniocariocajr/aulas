# Desafios de JavaScript

Bem-vindo à pasta de desafios! Aqui você encontrará exercícios práticos para fixar o conhecimento de cada nível.

## Como usar

1. Abra o arquivo de exercícios do nível que você está estudando.
2. Complete os desafios seguindo as instruções nos comentários.
3. Tente resolver sem olhar os exemplos da teoria primeiro!

- **Avançado:** Consumir uma API externa e exibir os dados de forma performática com Debounce.
